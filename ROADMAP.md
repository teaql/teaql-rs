# TeaQL Project Roadmap

Our vision is to provide the most robust, AI-friendly, and type-safe data runtime in Rust.

## Current Focus
- Enhance integration with LLM and coding agents.
- Improve streaming queries and multi-database execution paths.
- Increase unit and integration test coverage across all provider crates.

## Near-Term Goals (Next 6 Months)
- Achieve OpenSSF Best Practices Silver and Gold badges.
- Expand documentation and quick start guides.
- Implement robust telemetry and distributed tracing (OpenTelemetry).

## Long-Term Vision
- Build a fully declarative, model-driven ecosystem.
- Stabilize the runtime API for a 1.0 release.
